# 🤖 Cognix AI Chatbot

Cognix is a modern AI-powered chatbot that provides **quick, accurate, and concise answers** for General Knowledge (GK), Mathematics, and Logical Reasoning questions. It features a clean chat interface, real-time responses, and a smooth user experience.

---

## 🚀 Features

* 💬 Real-time AI chat interface
* 📚 Supports **General Knowledge (GK)** questions
* ➗ Solves **Mathematics problems**
* 🧠 Handles **Logical reasoning questions**
* ⚡ Fast responses using **Gemini API**
* ⌨️ Auto-resizing input field
* 🔄 Typing indicator animation
* ❌ Error handling for failed responses

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **API:** Google Gemini API
* **Tools:** VS Code, Git, GitHub

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/wanisanket12/Congnix-AI-Chatbot.git
cd Congnix-AI-Chatbot
```

---

### 2️⃣ Open project

Just open `index.html` in your browser
OR use Live Server in VS Code

---

### 3️⃣ Add API Key

In your JavaScript file:

```js
const apiKey = "YOUR_API_KEY";
```

> ⚠️ **Important:** Do not expose API keys in production. Use a backend server.

---

## 🧠 How It Works

* User enters a question
* The input is sent to **Gemini API**
* AI processes the request
* Response is displayed in chat format

---

## 🎯 Supported Question Types

### 📚 General Knowledge

* Capitals, history, science, etc.

### ➗ Mathematics

* Arithmetic, percentages, calculations

### 🧩 Logical Reasoning

* Patterns, sequences, puzzles

---

## 🔐 Security Note

Currently, the API key is used in the frontend.
For production:

* Move API calls to backend (Node.js / Django)
* Use environment variables
